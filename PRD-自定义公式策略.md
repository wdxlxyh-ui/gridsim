# PRD：自定义公式自动变化策略

> 版本：v2.2.0-dev  
> 日期：2026-05-15  
> 状态：待评审

---

## 1. 需求概述

为 IEC104 模拟器新增第 11 种自动变化策略——**自定义公式**。用户可选择多个关联测点，通过按钮组合构造一个包含 `+`、`-`、`*`、`/` 四则运算的公式，模拟器按周期计算公式结果并写入目标测点。

## 2. 用户故事

> 作为一个变电站自动化测试人员，我需要在模拟器中配置一个测点，其值由其他多个测点的值通过自定义公式计算得出（例如：母线电压 = 线路1电压 + 线路2电压 - 线路3电压），以便在 IEC104 客户端观察联动变化效果。

## 3. 功能规格

### 3.1 策略名称

- **中文**：自定义公式
- **英文代号**：`custom`（StrategyType 新增 `StrategyCustomFormula`）

### 3.2 参数定义

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `custom_ioas` | `string` | 是 | 关联测点 IOA 列表，分号分隔，最多 50 个，最少 2 个 |
| `custom_formula` | `string` | 是 | 公式字符串，由运算符和参数占位符组成 |
| `period_ms` | `int` | 是 | 计算周期（毫秒），≥100 |

### 3.3 参数 `{n}` 占位符规则

- `{0}` 表示第 1 个关联 IOA 的**当前值**
- `{1}` 表示第 2 个关联 IOA 的**当前值**
- ……
- `{n}` 表示第 n+1 个关联 IOA 的**当前值**
- 公式中可混合使用运算符和占位符
- 示例：`{0}+{1}-{2}` 表示第1个IOA值 + 第2个IOA值 − 第3个IOA值

### 3.4 公式约束

| 约束 | 说明 |
|------|------|
| 最少关联测点 | 2 个 |
| 最多关联测点 | 50 个 |
| 支持运算符 | `+`、`-`、`*`、`/` |
| 支持括号 | `(`、`)`（用于优先级控制） |
| 不支持函数 | 不支持 sin/cos/log 等数学函数 |
| 公式合法性 | 运行时解析，语法错误则本次计算跳过并记录日志告警 |
| 除零保护 | 发生除零时，该次计算跳过，结果保持上一次值 |

### 3.5 前端交互流程

```
┌─────────────────────────────────────────────┐
│  步骤一：选择关联测点                         │
│  ┌───────────────────────────────────────┐   │
│  │ 搜索/筛选测点列表                       │   │
│  │ ☑ IOA 1001 - 母线A相电压              │   │
│  │ ☑ IOA 1002 - 母线B相电压              │   │
│  │ ☑ IOA 1003 - 母线C相电压              │   │
│  │                                        │   │
│  │ 已选 3 个测点（最少2个，最多50个）       │   │
│  └───────────────────────────────────────┘   │
│                                                │
│  步骤二：构造公式（按钮点击方式）              │
│  ┌───────────────────────────────────────┐   │
│  │  公式显示区：{0}+{1}-{2}               │   │
│  │                                        │   │
│  │  [+][-][*][/][(][)]                   │   │
│  │  [{0}][{1}][{2}]  ← 按名称展示        │   │
│  │  [退格] [清空]                        │   │
│  └───────────────────────────────────────┘   │
│                                                │
│  步骤三：设置计算周期（默认 1000ms）           │
│  ┌───────────────────────────────────────┐   │
│  │  周期(ms): [1000 ▼]                   │   │
│  └───────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### 3.6 计算逻辑

```
每个 period_ms 触发一次：
  1. 遍历 custom_ioas 列表，依次读取每个 IOA 的当前 Value
  2. 将公式中的 {0}、{1}... 替换为实际值
  3. 使用 Go 标准库 govaluate（或手写简单解析器）计算表达式结果
  4. 将结果写入目标 IOA
  5. 触发 IEC104 变化上送（COT=3）
```

### 3.7 安全与容错

| 场景 | 处理方式 |
|------|----------|
| 关联测点不存在 | 配置时校验，拒绝保存 |
| 关联测点为 AO/DO 类型 | 允许关联，可读取其值 |
| 公式语法错误 | 运行时跳过本次计算，记录 error log |
| 除零错误 | 跳过本次计算，保持上一次值，记录 warning log |
| 运行时关联测点被删除 | 计算时动态校验，跳过不存在的 IOA |
| 周期 < 100ms | 配置和运行时均拒绝 |

## 4. API 变更

### 4.1 新增策略类型

`StrategyParams` 新增字段：

```json
{
  "custom_ioas": "16385;16386;16387",
  "custom_formula": "{0}+{1}-{2}"
}
```

### 4.2 自动变化配置接口（不变）

复用现有接口：

```
PUT /api/v1/instances/{id}/points/{ioa}/auto-change
{
  "strategy": "custom",
  "enabled": true,
  "params": {
    "custom_ioas": "16385;16386;16387",
    "custom_formula": "{0}+{1}-{2}",
    "period_ms": 1000
  }
}
```

### 4.3 查询时返回所有关联测点信息

```
GET /api/v1/instances/{id}/points/{ioa}/auto-change

Response:
{
  "ioa": 16388,
  "strategy": "custom",
  "enabled": true,
  "params": {
    "custom_ioas": "16385;16386;16387",
    "custom_formula": "{0}+{1}-{2}",
    "period_ms": 1000
  },
  "updated_at": "2026-05-15T10:00:00Z"
}
```

## 5. CSV 导入导出

### 5.1 导出格式

策略代码 `11` = 自定义公式

| 列 | 值 |
|----|------|
| 自动变化模式 | 11 |
| A列 | 关联IOA列表（分号分隔） |
| B列 | 公式字符串 |
| C列 | 周期(ms) |
| D~G列 | 空 |

### 5.2 导入格式

读取 A~C 列：
- A → `custom_ioas`
- B → `custom_formula`
- C → `period_ms`

## 6. 前端界面要求

### 6.1 弹窗模式

在现有自动变化弹窗中新增 Tab 页："自定义公式"

### 6.2 区域划分

| 区域 | 内容 |
|------|------|
| 关联测点选择 | 带搜索的多选下拉/表格，限制 2~50 个 |
| 公式构造区 | 按钮式公式编辑器，实时显示公式文本 |
| 参数说明 | 列出所选测点编号与名称对照表 |
| 周期设置 | 数字输入框，默认 1000ms |
| 预览 | 实时显示公式展开后的样子（如 `母线A相电压(235.5) + 母线B相电压(236.1) - 母线C相电压(234.8)`） |

### 6.3 按钮设计

两行按钮区域：

**第一行——运算符按钮：**  
`+`、`-`、`*`、`/`、`(`、`)`、`←`（退格）、`C`（清空）

**第二行——参数按钮：**  
根据所选关联测点动态生成，每个按钮显示测点名称和编号：
```
[母线A相电压(1001)]  [母线B相电压(1002)]  [母线C相电压(1003)]
```

点击按钮即在公式显示区追加对应文本。
