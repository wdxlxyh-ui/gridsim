什么是 GridSim？

GridSim 是一个 生产级 IEC104 从站模拟器，用于变电站自动化测试。支持多实例并行运行，模拟 RTU 及间隔层设备，适用于 SCADA 系统开发和集成测试。

自 v2.0 起引入多实例服务模式 + Vue 3 Web 界面，v2.4 起支持 Modbus TCP 协议并行运行，v2.5 新增微电网仿真模块。目前已迭代至 v2.5.3。

背后技术栈：Go 1.21+ 后端 + Vue 3 + TypeScript + Element Plus + ECharts 前端，纯静态编译单二进制部署。

核心能力

从一个模拟器到一个仿真平台 — GridSim 不断进化

⚡

IEC 104 协议引擎

完整实现 IEC104 从站协议：遥测 AI、遥信 DI、遥脉 PI、遥控 DO、遥调 AO。支持总召唤、电度召唤、变化上送 (COT=3)、品质描述 QDS。

🔌

Modbus TCP 并行运行

与 IEC104 共享同一数据源，支持功能码 01/02/03/04/05/06/15/16。点表扩展列自动识别，支持 4 种字节序（ABCD/CDAB/BADC/DCBA）。

🔄

11 种自动变化策略

递增、随机、CSV 回放、MAX/MIN、SOC 计算、电量统计、AO 关联、接口更新、手动、自定义公式。每个策略独立 goroutine 调度，实例启停自动管理。

📊

ECharts 实时趋势

TrendPage 全面重构，基于测点真实时间戳渲染。缩放拖拽、十字准线 tooltip、8 色主题、图例显隐。时间范围 5m~2h，轮询 200ms~5s 可配置。

📡

CSV 多测点同步回放

上传多列 CSV 文件，每列映射到不同测点。相对/绝对时间两种模式，多个测点共享同一时间基准严格同步。支持文件下拉选择和列名自动匹配。

🧠

MCP Server（28 工具）

Model Context Protocol 服务，提供完整程序化控制接口。实例管理（9 工具）、数据接口（17 工具）、全局工具（2 工具）。AI 助手可直接控制模拟器。

🔬

微电网仿真模块

设备管理 + SVG拓扑 + 6-step tick 仿真引擎。支持光伏、储能、负荷、充电桩四种设备。功率平衡公式、SOC 计算、远方/本地控制模式、SVG 拓扑图。

⚙️

多实例管理（最多 1000）

serve 子命令启动服务模式，通过 Vue 3 Web 界面管理实例生命周期。增删改查、启停控制、运行监控（自动刷新）、配置持久化。

典型应用场景

从变电站仿真到微电网研究，GridSim 无处不在

01

变电站遥测仿真

模拟 RTU 设备，通过 IEC104 协议向 SCADA 系统发送遥测/遥信数据。支持 HTTP API 动态修改值，实时触发变化上送 (COT=3)。

02

SCADA 集成测试

无需真实硬件，在开发环境中模拟上百个测点。多实例并行运行模拟不同变电站，验证 SCADA 系统的总召、遥控、变化上报处理逻辑。

03

微电网仿真研究

表单式设备管理，配置 PV/储能/负荷/充电桩。功率平衡自动计算，SOC 动态更新。适合微电网控制策略验证和能量管理系统测试。

04

AI Agent 工具链集成

通过 MCP Server 暴露 28 个工具，AI 助手可以直接操作模拟器。创建实例、读写测点、配置策略、控制 CSV 回放 — 全链路 AI 驱动自动化测试。

05

多规约网关验证

同一数据源同时提供 IEC104 和 Modbus TCP 接口，验证协议网关的数据一致性。适用多规约转换设备的出厂测试和验收。

06

CSV 录波数据回放

将真实故障录波 CSV 文件导入模拟器，多测点同步回放。用于保护装置测试、故障分析复现、算法验证。
