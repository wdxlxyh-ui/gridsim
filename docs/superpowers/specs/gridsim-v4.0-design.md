# GridSim v4.0 设计规范

## 设计概述

**目标**: 将 GridSim 从工业工具升级为专业 SCADA 仿真平台
**原则**: 专业优先、一致性、效率、可维护性
**核心用户**: SCADA 测试工程师（日活）、AI Agent

## 设计系统

- 深色主题: bg #0b1120, card #141b2d, text #e2e8f0
- 强调色: #3b82f6 (accent)
- 状态色: 运行 #22c55e, 警告 #f59e0b, 危险 #ef4444, 储能 #409eff
- 字体: 系统默认 + 'JetBrains Mono' (mono)
- 圆角: 卡片 12px, 按钮/输入框 6px
- 阴影: 卡片 0 4px 20px rgba(0,0,0,0.4)

## 布局

- Header 48px (品牌 + 面包屑 + 状态 + 搜索 + 用户)
- Sidebar 64px (悬停展开至 200px)
- 内容区自适应
- 路由: /dashboard, /config, /monitor, /trend, /proxy, /detail/:id, /microgrid/:id

## 页面改造

| 页面 | 改造内容 |
|------|---------|
| 仪表盘(新增) | 统计卡片 + 活跃实例 + 系统状态 + 活动日志 |
| 配置管理 | 卡片/表格双视图 | 实例卡片直接操作 |
| 微电网编辑 | 拆分9子组件, SVG深色风, 能流动画, 功率数据统一 |
| 实例详情 | 表格分组, 类型过滤, 右键菜单 |
| 趋势页 | 图标统一, 时间快捷选择, 面板拖拽 |
| 接口测试 | 拆分组件, 图标统一, JSON 格式化 |

## 微电网修复

- P0: 功率数据不一致(Dashboard复用power balance)
- P0: SVG拓扑深色化
- P0: 无功功率模型(power_factor配置)
- P1: 储能效率计算(Eff)
- P1: PV日照曲线
- P2: 能流动画, 仿真速度控制

## 实施计划

Phase1(设计系统+组件拆分 40h) -> Phase2(页面重设计 60h) -> Phase3(微电网修复 20h) -> Phase4(微交互 16h)
总计 ~136h
