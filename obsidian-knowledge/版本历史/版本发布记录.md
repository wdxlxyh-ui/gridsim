# 版本发布记录

## v2.5.x 系列

### v2.5.3 — 2026-05-22
- **ECharts 实时趋势重构**：TrendPage 全面改用 ECharts
  - 时间轴基于测点真实 `updated_at` 时间戳
  - 缩放拖拽 + 十字准线 tooltip
  - 8 色主题配色
- **批量读取 API**：`GET /points/batch?ioas=X,Y,Z` — N 次请求→1 次
- **崩溃修复**：engine.go goroutine nil pointer panic
- **构建改进**：内置 `bin/VERSION` + `config/users.json`

### v2.5.2 — 2026-05-21
- CSV 文件下拉选择（共享目录 + 实例私有目录）
- 多列 CSV 自动按列名匹配 IOA
- 新增 `list_csv_files` MCP 工具
- 修复前端登录路径、MCP write_points schema

### v2.5.1 — 2026-05-20
- ConfigPage 顶部显示实例容量（已配置/1000、运行中、已停止）

### v2.5.0 — 2026-05-20
- **CSV 多测点同步回放**：多列 CSV 映射、相对/绝对时间、状态持久化
- CSV 回放时序基于 elapsed time 控制

## v2.4.x 系列

### v2.4.0 — 2026-05-19
- **Modbus TCP 协议支持**：功能码 1/3/4/5/6/16
- CSV 回放时序控制修复
- 实例上限提升至 1000

## v2.2.x 系列

### v2.2.0 — 2026-05-15
- **自定义公式策略**（第 11 种）
  - 按钮式公式编辑器（+ - * / ( )）
  - `{n}` 占位符，2~50 个关联测点
  - 递归下降解析器 + 除零保护
- TrendPage 多线 SVG 趋势对比
- CSV 上传路径修复
- 用户登录方案设计
- AOFollow 回调机制修复

## v2.1.x 系列

### v2.1.5 — 2026-05
- 修复 Web 前端路径（基于可执行文件定位）
- 修复 Makefile brace expansion 打包问题

### v2.1.4 — 2026-05
- 置数框缺少 @input 事件 → 修复
- DI 开关状态不更新 → 修复
- 置数与策略冲突导致禁用 → 修复
- 手动置数策略 UI 支持

### v2.1.3 — 2026-05
- **手动策略**（第 10 种）：`manual`，引擎不自动计算
- 导出/导入 CSV 支持 code 10

### v2.1 — 2026-05
- **实例详情页**：测点高频刷新、置数
- **自动变化引擎**（10 种策略）
- 批量配置、导出/导入、CSV 上传
- 置数隔离存储
