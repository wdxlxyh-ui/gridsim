# 数据模型

## Point（测点）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ioa` | uint32 | IOA 地址，全库唯一 |
| `name` | string | 测点名称 |
| `value_type` | string | `FLOAT` / `DOUBLE` / `INT` / `BIT` |
| `point_type` | string | `AI` / `DI` / `PI` / `DO` / `AO` |
| `value` | float64 | 浮点值（AI/AO 使用） |
| `bool_value` | bool | 布尔值（DI/DO 使用） |
| `int_value` | int32 | 整型值（PI 使用） |
| `efficient` | float64 | 系数 |
| `base_value` | float64 | 基值（初始值） |
| `qds` | QualityDescriptor | 品质描述 |
| `alias` | string | 别名 |
| `timestamp` | time.Time | 更新时间戳 |
| `function_code` | uint8 | Modbus 功能码 |
| `register_address` | uint16 | Modbus 寄存器地址 |
| `byte_order` | string | Modbus 字节序 |

### PointType 映射

| PointType | 中文 | IEC104 ASDU TypeID |
|-----------|------|---------------------|
| AI | 遥测 YC | M_ME_NC_1 (13) |
| DI | 遥信 YX | M_SP_NA_1 (1) |
| PI | 遥脉 YM | M_IT_NA_1 (15) |
| DO | 遥控 | C_SC_NA_1 (45) |
| AO | 遥调 | C_SE_NC_1 (48) |

### QualityDescriptor（品质描述）

| 字段 | 类型 | 说明 |
|------|------|------|
| `invalid` | bool | 无效 |
| `not_topical` | bool | 非当前 |
| `substituted` | bool | 取代/替代 |
| `overflow` | bool | 溢出 |
| `blocked` | bool | 闭锁 |

## AutoChangeConfig（自动变化配置）

> 存储位置：`config/auto_changes/{instance_id}.json`

| 字段 | 类型 | 说明 |
|------|------|------|
| `ioa` | uint32 | 目标测点 IOA |
| `strategy` | string | 策略类型 |
| `enabled` | bool | 是否启用 |
| `params` | StrategyParams | 策略参数 |
| `updated_at` | time.Time | 更新时间 |

### StrategyParams（策略参数）

参数结构因策略类型而异，详见 [[功能特性/自动变化策略|自动变化策略]]。

## PointSnapshot（实时测点快照）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ioa` | uint32 | IOA 地址 |
| `name` | string | 测点名称 |
| `point_type` | string | AI/DI/PI/DO/AO |
| `value` | float64 | 浮点值 |
| `bool_value` | bool | 布尔值 |
| `int_value` | int32 | 整型值 |
| `updated_at` | time.Time | 毫秒精度时间戳 |
| `unit` | string | 单位 |

## InstanceConfig（实例配置）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | 否 | 实例 ID（自动生成） |
| `name` | string | 是 | 实例名称 |
| `iec104_port` | int | 是 | IEC104 端口 |
| `xlsx_file` | string | 是 | 点表文件路径 |
| `enabled` | bool | 否 | 是否启用 |
| `protocol` | string | 否 | 规约类型（默认 iec104） |
| `modbus_config` | object | 否 | Modbus 配置 |

## InstanceState（实例状态）

| 字段 | 类型 | 说明 |
|------|------|------|
| `status` | string | `running` / `stopped` / `error` |
| `stats.uptime_seconds` | int64 | 运行时长 |
| `stats.total_points` | int | 测点总数 |
| `stats.client_connected` | bool | 客户端连接状态 |
| `stats.interrogations` | int64 | 总召次数 |
| `stats.controls` | int64 | 遥控次数 |
| `stats.spontaneous` | int64 | 变化上送次数 |

## Store（内存点表）

```go
type Store struct {
    mu     sync.RWMutex
    points map[uint32]*Point       // IOA 索引
    byType map[PointType][]*Point  // 类型索引
}
```

支持操作：
- `Get(ioa)` / `SetValue(ioa, value)` / `Delete(ioa)`
- `GetByType(pointType)` （用于总召分组）
- `GetByFunctionCodeAndAddress(fc, addr)` （Modbus 寻址）
- `GetByFunctionCodeRange(fc, startAddr, count)` （Modbus 范围读取）
