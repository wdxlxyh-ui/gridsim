# GridSim — 多协议电网仿真平台

> [!note] 项目简介
> **IEC104 从站模拟器**，用于变电站自动化测试。支持多实例并行运行，模拟 RTU 及间隔层设备，适用于 SCADA 系统开发和集成测试。

| 属性 | 值 |
|------|-----|
| 当前版本 | **v2.5.3** (2026-05-22) |
| 编程语言 | Go 1.21+ |
| 前端 | Vue 3 + TypeScript + Element Plus + ECharts |
| 协议支持 | IEC104, Modbus TCP |
| 运行模式 | 传统单实例 + 服务多实例 |
| 最大实例数 | 1000 |
| 许可证 | MIT |

## 核心特性

- **双运行模式** — 传统模式（单进程 CLI）和服务模式（多实例 + Web 管理界面）
- **Excel 点表配置** — 在 `.xlsx` 中定义 IOA、点类型、系数、初始值
- **完整 IEC 104 协议** — 遥测 AI、遥信 DI、遥脉 PI、遥控 DO、遥调 AO
- **Modbus TCP 协议** — 自 v2.4.0 起支持，与 IEC104 并行运行
- **RESTful HTTP API** — 查询、单点/批量更新、修改 QDS
- **11 种自动变化策略** — 递增、随机、CSV 回放、MAX、MIN、SOC、电量、AO 关联、接口更新、手动、自定义公式
- **MCP Server** — 28 个工具，支持 AI 助手直接控制模拟器
- **ECharts 实时趋势** — 多线对比、缩放拖拽、十字准线
- **CSV 多测点同步回放** — 多列 CSV 映射到不同测点，严格同步
- **微电网仿真** — 列表式设备管理 + SVG拓扑图 + 实时仿真引擎（PV/储能/负荷/充电桩），自动公式 + SOC 计算
- **跨平台编译** — Linux amd64/arm64、Windows amd64、`.deb` 打包
