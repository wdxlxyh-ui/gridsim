# EGC AI 自动化测试方案

> 版本：v3.0

## 概述

基于 AI 测试用例生成 + 双 MCP 服务架构 + IEC104 模拟器，实现从触发执行→环境准备→逐用例循环校验→报告输出→归档留痕的全链路闭环回归测试。

## 整体架构

```
触发 → 环境准备 → 逐用例循环校验 → 报告归档
```

## 核心闭环（4 步）

### 步骤 1：触发 & 指定场景
| 方式 | 说明 |
|------|------|
| 定时任务 | 每日凌晨/每周全量回归 |
| 手动触发 | 指定场景名称 |
| 事件驱动 | EGC 版本发布后自动触发 |

### 步骤 2：环境准备
- EGC 控制策略配置
- 测试数据生成（CSV 时序数据）
- 环境就绪检查

### 步骤 3：逐用例循环校验
```
① 写入输入数据 → ② 等待 EGC 控制周期
 → ③ 读取控制结果 → ④ 比对 & 记录
```

### 步骤 4：分析 → 报告 → 归档
- 生成报告（通过率、明细、分布）
- 推送报告
- 全量归档
- 回归对比

## 双 MCP 服务

| 服务 | 功能 |
|------|------|
| Instance Manager | 管理面：实例生命周期 |
| Data Interface | 数据面：读写/策略/CSV/QDS |
