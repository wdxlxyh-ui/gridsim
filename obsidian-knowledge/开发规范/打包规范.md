# 打包规范

## Makefile 规范

| 规范 | 说明 | 反例 |
|------|------|------|
| `mkdir` 逐行创建 | `/bin/sh` 不支持 `{a,b,c}` 展开 | `mkdir -p dir/{bin,config}` ❌ |
| amd64 无后缀 | 输出 `bin/gridsim` | `bin/gridsim-linux-amd64` ❌ |
| arm64 加 `-arm64` | 输出 `bin/gridsim-arm64` | |
| Windows 加 `.exe` | 输出 `bin/gridsim.exe` | |
| `dist` 依赖顺序 | 先前端构建，再打包二进制 | |

## 发行包结构

```
gridsim-v{version}-linux-amd64/
├── bin/
│   ├── gridsim
│   ├── iec104-mcp
│   ├── start.sh / stop.sh / restart.sh
├── config/
│   └── instances.json
├── logs/
├── resources/
└── web/dist/
    ├── index.html
    └── assets/
```

## 前端路径解析

```go
// 二进制通过 filepath.Dir(exePath) 定位 bin/ 目录
// 前端路径：
filepath.Join(filepath.Dir(exePath), "..", "web", "dist")
```

> ⚠️ `cp -r web/dist $(TARGET)/web` 会将 dist **内容** 直接放到 web/ 下，导致路径不匹配。**正确做法**：
> ```makefile
> mkdir -p $(TARGET)/web/dist && cp -r web/dist/* $(TARGET)/web/dist/
> ```

## 编译目标

| 目标 | 架构 |
|------|------|
| Linux amd64 | 通用 Linux（Debian/Ubuntu/银河麒麟） |
| Linux arm64 | ARM 服务器 |
| Windows amd64 | Windows |
| .deb | Debian 系列 |
| UPX 压缩 | 减少约 60% 体积 |
