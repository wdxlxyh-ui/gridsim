# Go 后端规范

## 路径处理

```go
// ✅ 正确
webDir := filepath.Join(filepath.Dir(exePath), "..", "web", "dist")

// ❌ 禁止：硬编码斜杠
webDir := exeDir + "/../web/dist"
```

## 错误处理

```go
// ✅ 正确
if err := cfgStore.Load(); err != nil {
    slog.Warn("加载配置失败，使用空配置", "error", err)
}

// ❌ 禁止：忽略错误
cfgStore.Load()

// ❌ 禁止：panic
```

## 路由注册

```go
// ✅ 正确：通过 Register 方法注册
func (h *DetailHandler) Register(mux *http.ServeMux) {
    mux.HandleFunc("/api/v1/instances/", h.handleInstanceByID)
}

// ❌ 禁止：全局变量直接注册
```

## 策略模式

```go
// ✅ 正确：空的策略逻辑
case model.StrategyManual:
    // 空逻辑，值由用户 API 置数

// ✅ 正确：允许写入的策略
return cfg.Strategy == model.StrategyAPIUpdate || cfg.Strategy == model.StrategyManual
```

## 文件上传与 CSV 路径

- 上传文件保存到 `configDir` 目录（非硬编码 `./config/`）
- CSV 上传路径：`configDir/csv/{instanceID}/{filename}`
- 查找顺序：实例目录 → 共享目录

## AOFollow 架构

```
IEC104 遥控 → Store.SetValue(AO) → 引擎.HandleAOFollow(aoIOA)
  → 找到所有 FollowAOIOA 匹配的 AI 点 → 复制值 → Publish
```

> 自 v2.2.0 起修复：`iec104.Server` 增加 `aoFollowFn` 回调
