# 自定义公式策略

> 自 v2.2.0 起支持

## 概述

第 11 种自动变化策略。用户选择多个关联测点，通过按钮式编辑器构造四则运算公式，模拟器按周期计算公式结果并写入目标测点。

## 前端交互

```
步骤一：选择关联测点（2~50 个）
步骤二：按钮式构造公式（运算符+参数按钮）
步骤三：设置计算周期（≥100ms）
```

## 公式编辑器 UI

| 元素 | 说明 |
|------|------|
| 关联测点选择 | 带搜索的多选下拉，限制 2~50 个 |
| 公式构造区 | 按钮式编辑器 |
| 参数说明 | 所选测点编号与名称对照表 |
| 周期设置 | 数字输入框，默认 1000ms |
| 预览 | 公式展开后的样子（如 `母线A相电压(235.5) + 母线B相电压(236.1)`） |

## 按钮设计

```
第一行（运算符）:  +  -  *  /  (  )  ←(退格)  C(清空)
第二行（参数）:   动态生成关联测点按钮 {0} {1} {2} ...
```

## 实现技术

- 递归下降表达式解析器
- 支持括号优先级
- 除零保护
- 运行时公式语法校验

## 用户故事

> 作为一个变电站自动化测试人员，我需要在模拟器中配置一个测点，其值由其他多个测点的值通过自定义公式计算得出（例如：母线电压 = 线路1电压 + 线路2电压 - 线路3电压），以便在 IEC104 客户端观察联动变化效果。
