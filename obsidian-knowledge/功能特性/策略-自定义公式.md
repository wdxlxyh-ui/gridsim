# 自定义公式策略

英文代号：`custom`

> 自 v2.2.0 起引入，第 11 种策略

## 说明

支持通过四则运算公式对多个关联测点进行计算，结果写入目标测点。

## 适用场景

复杂联动模拟，如"母线电压 = 线路1电压 + 线路2电压 - 线路3电压"。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `custom_ioas` | string | 是 | 关联 IOA 列表，分号分隔 |
| `custom_formula` | string | 是 | 公式字符串 |
| `period_ms` | int | 是 | 计算周期（毫秒，≥100） |

## `{n}` 占位符规则

- `{0}` = 第 1 个关联 IOA 的当前值
- `{1}` = 第 2 个关联 IOA 的当前值
- 以此类推
- 示例：`{0}+{1}-{2}`

## 公式约束

| 约束 | 说明 |
|------|------|
| 最少关联测点 | 2 个 |
| 最多关联测点 | 50 个 |
| 支持运算符 | `+` `-` `*` `/` `(` `)` |
| 不支持函数 | sin/cos/log 等 |

## 计算逻辑

```
每个 period_ms 触发一次:
  1. 遍历 custom_ioas，读取各 IOA 的 Value
  2. 替换 {0}、{1}... 为实际值
  3. 递归下降表达式解析器计算结果
  4. 写入目标 IOA
  5. 触发变化上送
```

## 安全与容错

| 场景 | 处理方式 |
|------|----------|
| 关联测点不存在 | 配置时校验，拒绝保存 |
| 公式语法错误 | 跳过本次计算，记录 error log |
| 除零错误 | 跳过，保持上次值，记录 warning |
| 周期 < 100ms | 配置和运行时均拒绝 |

## CSV 导入/导出

| 列 | 值 |
|----|-----|
| 策略代码 | 11 |
| A 列 | 关联 IOA 列表 |
| B 列 | 公式字符串 |
| C 列 | 周期(ms) |

## 配置示例

```json
{
  "strategy": "custom",
  "enabled": true,
  "params": {
    "custom_ioas": "16385;16386;16387",
    "custom_formula": "{0}+{1}-{2}",
    "period_ms": 1000
  }
}
```
