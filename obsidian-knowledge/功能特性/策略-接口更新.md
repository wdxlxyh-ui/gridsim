# 接口更新策略

英文代号：`apiupdate`

## 说明

仅允许 HTTP API 写入，引擎不做任何计算。引擎在计算时直接跳过。

## 适用场景

外部系统联调，值完全由外部系统通过 API 控制。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `api_init_value` | float64 | 否 | 初始值 |

## 计算逻辑

```
周期触发:
  // 空逻辑 — 引擎不计算
  // 值仅由 HTTP API / MCP write_point 写入
```

## 配置示例

```json
{
  "strategy": "apiupdate",
  "enabled": true,
  "params": {
    "api_init_value": 0
  }
}
```
