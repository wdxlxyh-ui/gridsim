# 电量统计策略

英文代号：`energy`

## 说明

基于功率积分计算累计电量。

## 适用场景

电能计量仿真，模拟电表累计电量。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `init_energy` | float64 | 是 | 初始电量（kWh） |
| `stat_type` | int | 是 | 0=充电 1=放电 |
| `energy_power_ioa` | uint32 | 是 | 功率测点 IOA |
| `energy_period_ms` | int | 是 | 积分周期（毫秒） |

## 计算逻辑

```
周期触发:
  power = store.Get(energyPowerIOA).Value
  energy = power * (energyPeriodMs / 3600000.0)
  
  if statType == 0:   // 充电
    if power > 0: totalEnergy += energy
  else:               // 放电
    if power < 0: totalEnergy += abs(energy)
  
  store.SetValue(ioa, totalEnergy)
  publisher.Publish(point)
```

## 配置示例

```json
{
  "strategy": "energy",
  "enabled": true,
  "params": {
    "init_energy": 0,
    "stat_type": 0,
    "energy_power_ioa": 16385,
    "energy_period_ms": 1000
  }
}
```
