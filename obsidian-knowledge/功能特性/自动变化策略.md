## 引擎架构

```
┌─────────────────────────────────┐
│         AutoChangeEngine        │
│─────────────────────────────────│
│  - tasks map[ioa]*ChangeTask    │
│  - store *AutoChangeStore       │
│  - pointStore *library.Store    │
│  - publisher                    │
└──────────┬──────────────────────┘
           │
    ┌──────┴──────┐
    │  ChangeTask  │  (goroutine per task)
    │──────────────│
    │  ticker      │
    │  calc()      │ ← Strategy calc
    │  write()     │ ← Store + Publish
    └──────────────┘
```

## 生命周期

| 动作 | 行为 |
|------|------|
| 实例启动 | 从 `auto_changes/{id}.json` 加载配置，启动 enabled 任务 |
| 实例停止 | cancel 所有 goroutine，保存状态 |
| 前端配置 | PUT API → 更新配置 → 重启对应任务 goroutine |
| 前端关闭 | DELETE API → 停止 goroutine → 删除配置 |

## 策略参数结构

```json
{
  "ioa": 16385,

  "enabled": true,
  "params": {

    "step": 1,
    "period_ms": 1000,
    "max_value": 100
  }
}
```
