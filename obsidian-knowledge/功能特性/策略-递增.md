# 递增策略

英文代号：`increment`

## 说明

每个周期为值加上步长，达到最大值后回到起始值。

## 适用场景

模拟缓慢上升的遥测量，如温度、压力等渐变过程。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `start_value` | float64 | 是 | 起始值 |
| `step` | float64 | 是 | 步长 |
| `period_ms` | int | 是 | 周期（毫秒，≥100） |
| `max_value` | float64 | 是 | 最大值（达到后回起始值） |

## 计算逻辑

```
周期触发:
  current = store.Get(ioa).Value
  next = current + step
  if next > max_value: next = start_value
  store.SetValue(ioa, next)
  publisher.Publish(point)
```

## 配置示例

```json
{
  "strategy": "increment",
  "enabled": true,
  "params": {
    "start_value": 0,
    "step": 1,
    "period_ms": 1000,
    "max_value": 100
  }
}
```
