# 手动策略

英文代号：`manual`

> 自 v2.1.3 起引入，第 10 种策略

## 说明

引擎不自动计算值，值完全由用户通过 API 手动置数。适用于外部系统联调场景。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 无 | — | — | 手动策略无需额外参数 |

## 计算逻辑

```
周期触发:
  // 空逻辑 — 引擎不计算
  // 值仅由用户 API 置数
```

## 与 apiupdate 的区别

| 策略 | 允许 API 写入 | 引擎行为 |
|------|--------------|----------|
| manual | ✅ | 空逻辑，完全跳过 |
| apiupdate | ✅ | 空逻辑，完全跳过 |

两者行为相同，manual 是更语义化的命名。

## 配置示例

```json
{
  "strategy": "manual",
  "enabled": true,
  "params": {}
}
```
