# CSV 多测点同步回放

> 自 v2.5.0 起支持

## 概述

在详情页新增独立"CSV 多测点回放"卡片，支持上传多列 CSV 文件，每列映射到不同测点，多个测点共享同一时间基准严格同步回放。

## CSV 文件格式

| time | 母线电压 | 线路电流 | 有功功率 |
|------|----------|----------|----------|
| 0 | 220.0 | 5.2 | 1144.0 |
| 1000 | 221.5 | 5.3 | 1173.9 |
| 2000 | 219.8 | 5.1 | 1120.9 |

## 两种时间模式

| 模式 | 说明 |
|------|------|
| **相对时间 (relative)** | 第 1 列为相对时间（ms 或 s），基于 elapsed time 控制播放节奏 |
| **绝对时间 (absolute)** | 时间格式为 `hh:mm:ss`，按系统时钟定位当前行 |

## 列映射

- 上传时自动按列名匹配测点名称
- 回退按序号分配（第 1 列 → IOA 列表第 1 个...）
- 支持从已有文件下拉选择（共享目录 + 实例私有目录）

## 版本增强

- **v2.5.0**: 初始实现，多列 CSV 映射，相对/绝对时间，状态持久化
- **v2.5.2**: CSV 文件下拉选择 + 自动按列名匹配 + `list_csv_files` MCP 工具
- **v2.5.3**: 无直接变更，但批量读取 API 优化了大点数场景的性能

## 向后兼容

单测点 CSV 模式完全不变，多测点回放为新增独立功能。
