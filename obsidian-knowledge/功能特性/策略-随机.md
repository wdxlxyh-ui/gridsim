# 随机策略

英文代号：`random`

## 说明

在指定的最小值和最大值范围内随机取值。

## 适用场景

模拟噪声或波动信号，如测量噪声、轻微波动的遥测量。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `min_value` | float64 | 是 | 最小值 |
| `max_value` | float64 | 是 | 最大值 |
| `decimal_places` | int | 否 | 小数位数（0 或 1） |
| `period_ms` | int | 是 | 周期（毫秒，≥100） |

## 计算逻辑

```
周期触发:
  value = min + rand.Float64() * (max - min)
  if decimalPlaces == 1: value = round(value, 1)
  else: value = round(value, 0)
  store.SetValue(ioa, value)
  publisher.Publish(point)
```

## 配置示例

```json
{
  "strategy": "random",
  "enabled": true,
  "params": {
    "min_value": 0,
    "max_value": 100,
    "decimal_places": 1,
    "period_ms": 1000
  }
}
```
