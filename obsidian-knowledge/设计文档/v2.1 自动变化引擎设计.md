# v2.1 自动变化引擎设计

## 概述

为运行中的实例提供详情页支持：测点实时值高频刷新、置数、自动变化策略、批量配置。

## 模块结构

```
internal/detail/
├── engine.go       自动变化调度引擎
├── strategy.go     策略计算逻辑
├── handler.go      详情页 HTTP API
└── store.go        配置持久化
```

## 引擎架构

```
┌─────────────────────────────────┐
│         AutoChangeEngine        │
│─────────────────────────────────│
│  - mu sync.RWMutex              │
│  - tasks map[ioa]*ChangeTask    │
│  - store *AutoChangeStore       │
│  - iecPub Publisher             │
│  - pointStore *library.Store    │
└──────────┬──────────────────────┘
           │
    ┌──────┴──────┐
    │  ChangeTask  │  (goroutine per task)
    │──────────────│
    │  ticker      │
    │  calc()      │ ← 调用 Strategy 计算
    │  write()     │ ← Store + Publish
    └──────────────┘
```

## 生命周期

| 动作 | 行为 |
|------|------|
| 实例启动 | 从 `auto_changes/{id}.json` 加载，启动所有 enabled 任务 |
| 实例停止 | cancel 所有 goroutine，保存状态 |
| 前端配置 | PUT → 更新配置 → 重启对应任务 goroutine |
| 前端关闭 | DELETE → 停止 goroutine → 删除配置 |

原始 v2.1 设计包含 9 种（不含 manual 和 custom），后续版本扩展至 11 种。
